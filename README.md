# RPG Character System

Um mini-projeto em Java que demonstra **programação orientada a objetos (POO)** com herança, polimorfismo e métodos exclusivos em subclasses.

---

## 📝 Descrição

O projeto possui uma **superclasse `Character`** e duas **subclasses**:
- `Warrior` – Guerreiro, com armas e armadura.
- `Mage` – Mago, com mana e magias.

O objetivo é **demonstrar polimorfismo**: métodos sobrescritos como `attack()` e `defense()` se comportam de forma diferente para cada subclasse, além de métodos exclusivos como:
- `heavyStrike()` → Warrior
- `castSpell()` → Mage

---

## ⚡ Funcionalidades

- Criar personagens com atributos básicos (`name`, `health`, `attack`, `defense`, `level`).
- Sobrescrever métodos de ataque e defesa para comportamentos específicos de cada classe.
- Aplicar dano (`takeDamage`) e cura (`heal`) nos personagens.
- Demonstrar polimorfismo armazenando diferentes personagens em uma coleção da superclasse.
- Métodos exclusivos para habilidades especiais.
